 HOW TO ASSEMBLE THE THESIS  (read me first)
==============================================================================

These drafts are written as LaTeX-ready content that maps 1:1 onto the sections
of main.tex. Integration is mostly copy-paste; nothing about the template's
formatting, packages or front-matter machinery needs to change.

------------------------------------------------------------------------------

------------------------------------------------------------------------------

                             \frontmatterheading{ABSTRACT} (~line 213).

NOTE ON DEPTH / PAGE COUNT: these are the expanded, deep-dive drafts written to
push the compiled thesis past 30 pages. Chapter 3 (Methodology) contains
pseudocode in \begin{verbatim} blocks -- verbatim is core LaTeX, so NO extra
package is needed. The verbatim lines are kept under ~72 characters so they fit
inside the 1in margins; if you change the geometry, re-check that none overflow.

NOTE ON CITATIONS: the bibliography now has 12 entries (ref1..ref12). New ones
beyond the original 8: ref9 DeBERTa (NLI verifier), ref10 Sentence-BERT (MiniLM
embeddings), ref11 scikit-learn, ref12 CBDT Annual Report (notice-predictor
feature basis). All are cited in the text -- don't drop any without removing its
\cite, or LaTeX will warn.

NOTE ON CHAPTER 4: the expanded challenges file adds a \section*{...} with an
\addcontentsline so the discussion shows in the TOC. This compiles as-is.

------------------------------------------------------------------------------
 TWO WAYS TO INTEGRATE
------------------------------------------------------------------------------
 OPTION A (simplest): open main.tex, delete each chapter's lipsum body, and
   paste the matching file's contents in its place.

 OPTION B (cleaner, recommended): keep main.tex as a thin shell and replace the
   chapter bodies with \input commands, e.g.:
        \input{chapters/02_introduction}
        \input{chapters/03_literature_survey}
        ...
   Put the .tex files in a chapters/ folder next to main.tex. For the abstract
   and bibliography, \input only the body (they are written to drop into the
   existing front-matter / thebibliography blocks).
